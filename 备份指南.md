## 💾 备份方法

### 方法1：手动打包（推荐）

```bash
# 进入工作区
cd /root/.openclaw/workspace

# 创建备份（包含所有重要文件）
tar -czf backup-$(date +%Y%m%d).tar.gz \
  *.md \
  *.json \
  trade-log.js \
  positions.js \
  review.js \
  btc-harami-monitor.js \
  memory/

# 查看备份文件
ls -lh backup-*.tar.gz
```

### 方法2：只备份数据文件

```bash
cd /root/.openclaw/workspace

# 只备份MD和JSON数据
tar -czf data-backup-$(date +%Y%m%d).tar.gz \
  *.md \
  *.json \
  memory/
```

### 方法3：同步到其他位置

```bash
# 复制到其他目录
cp -r /root/.openclaw/workspace /backup/workspace-$(date +%Y%m%d)

# 或使用rsync同步
rsync -av /root/.openclaw/workspace/ /backup/workspace/
```

你可以通过以下方式下载：

2. 通过文件管理器下载
3. 我可以帮你发送到 Telegram（如果文件不大）

## ⚠️ 最重要的文件

如果只能备份几个文件，优先备份这些：

1. **交易记录.json** - 所有交易数据
2. **复盘记录.json** - 所有复盘和教训
3. **MEMORY.md** - 长期记忆
4. **memory/** 文件夹 - 每日记忆

## 🔄 自动备份（可选）

可以设置定时备份：

```bash
# 每天凌晨2点自动备份
0 2 * * * cd /root/.openclaw/workspace && tar -czf /backup/workspace-$(date +\%Y\%m\%d).tar.gz *.md *.json memory/
```
