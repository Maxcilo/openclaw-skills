# Weather Trader 完整策略说明

## 📊 核心策略

### 1. 数据源
- **NOAA（美国国家海洋和大气管理局）天气预测**
- 准确率：~85%（1-2天预测）
- 数据来源：官方气象站（LaGuardia、O'Hare、Sea-Tac等）

### 2. 交易逻辑

#### 买入条件（全部满足）：
1. **NOAA 预测温度完全在市场温度区间内**
   - 例如：NOAA 预测 43°F，市场区间 42-43°F ✅
   - 例如：NOAA 预测 43°F，市场区间 44-45°F ❌（边缘，跳过）

2. **市场价格 < 入场阈值（10%）**
   - 市场价格 = 市场认为的概率
   - 如果市场只给 8% 概率，但 NOAA 说 85% → 买入机会

3. **价格在可交易范围内**
   - 价格 > $0.01（最低价格）
   - 价格 < $0.99（最高价格）
   - 避免极端价格市场

4. **通过安全检查**
   - 滑点 < 15%
   - 距离结算 > 2 小时
   - 单市场仓位 < $30

5. **未达到交易限制**
   - 每次运行最多 5 笔交易
   - 单市场最大仓位 $30
   - 单次交易最大 $2（或智能仓位）

#### 卖出条件：
1. **市场价格 ≥ 退出阈值（25%）**
   - 买入价 8% → 卖出价 25% = 盈利 212%
   
2. **持有股数 ≥ 5 股**（Polymarket 最低要求）

3. **通过安全检查**（同买入）

### 3. 风险控制

#### 仓位管理：
- **单次交易：** $1-2（默认）
- **智能仓位：** 账户余额的 5%（可选）
- **单市场最大：** $30
- **每次运行最多：** 5 笔交易

#### 安全机制：
- **滑点保护：** 超过 15% 跳过
- **时间保护：** 结算前 2 小时停止交易
- **价格趋势检测：** 24 小时内下跌 >10% = 更强信号
- **市场上下文检查：** 流动性、交易量、价格稳定性

### 4. 支持的城市
- NYC（纽约 - LaGuardia）
- Chicago（芝加哥 - O'Hare）
- Seattle（西雅图 - Sea-Tac）
- Atlanta（亚特兰大 - Hartsfield）
- Dallas（达拉斯 - DFW）
- Miami（迈阿密 - MIA）

## 📈 策略优势

1. **数据可靠：** NOAA 是官方气象机构，准确率高
2. **逻辑简单：** 只交易高确定性的（预测温度在区间内）
3. **风险可控：** 多重安全检查，仓位限制
4. **自动化：** 每 10 分钟自动运行，无需人工干预

## 🔧 可调整参数

你可以通过环境变量或 config.json 调整：

```bash
SIMMER_WEATHER_ENTRY=0.10        # 入场阈值（默认 10%）
SIMMER_WEATHER_EXIT=0.25         # 退出阈值（默认 25%）
SIMMER_WEATHER_MAX_POSITION=2.00 # 单次交易（默认 $2）
SIMMER_WEATHER_MAX_MARKET_EXPOSURE=30.00  # 单市场最大（默认 $30）
SIMMER_WEATHER_MAX_TRADES=5      # 每次最多交易（默认 5）
SIMMER_WEATHER_LOCATIONS="NYC,Chicago"  # 活跃城市
```

## 💡 策略核心思想

**"当专业气象预测显示高概率，但市场价格低估时 → 买入"**

这是一个**信息不对称套利**策略：
- NOAA 有专业气象模型和实时数据
- 市场参与者可能没有查看 NOAA 预测
- 当两者出现偏差时 → 套利机会
