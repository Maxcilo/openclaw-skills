# Memory Optimizer

记忆结构优化任务：定期压缩/精简日记文件，备份后处理，保护敏感信息。

## 功能

- 定期清理过时日记
- 保护敏感信息（钱包地址、API Key 等）
- 验证核心内容完整性
- 自动备份机制

## 使用方式

触发词：**记忆优化**、**压缩日记**、**memory optimize**、**clean memory**

## 执行前提

- 仅处理 **1天前** 的日记文件
- 必须在处理前 **先备份**
- 必须保护敏感信息不被删除
