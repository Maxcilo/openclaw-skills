# Web Fetch Skill

智能网页抓取技能，支持微信公众号、推特、普通网页。

## 版本

v1.4.0

## 功能

- 🤖 AI 摘要生成
- 📝 聊天窗口显示
- 🔢 准确字数统计
- 💾 MD + PDF 双输出
- 📱 Telegram 集成
- 🖼️ 图片嵌入
- 🎯 智能路由

## 安装

```bash
pip install -r requirements.txt
```

## 使用

```bash
# AI 摘要版（推荐）
python3 web_fetch_with_summary.py <url>

# PDF 版
python3 web_fetch_pdf.py <url>

# 图片嵌入版
python3 web_fetch_embedded.py <url>

# 增强版
python3 web_fetch_enhanced.py <url>

# 基础版
python3 web_fetch.py <url>
```

## 文件说明

- `SKILL.md` - 技能文档
- `web_fetch_with_summary.py` - AI 摘要版（推荐）
- `web_fetch_pdf.py` - PDF 生成版
- `web_fetch_embedded.py` - 图片嵌入版
- `web_fetch_enhanced.py` - 增强版
- `web_fetch.py` - 基础版
- `scrapling_fetch.py` - Scrapling 抓取脚本
- `requirements.txt` - 依赖列表
