## ⚠️ 注意事项

### 1. 流动性风险
```
刷量时需要确保：
- 市场有足够流动性
- 买卖价差小
- 能立即成交
```

### 2. 滑点风险
```
如果买卖价差>0.1%：
- 实际磨损会更高
- 需要选择流动性好的市场
```

### 3. 时间成本
```
刷10次需要：
- 每次2-3分钟
- 总计20-30分钟
```

### 4. 平台规则
```
检查Polymarket是否有：
- 刷量检测
- 账户限制
- 手续费变化
```

## 💡 降低磨损的技巧

### 技巧1：选择流动性好的市场
```
优先选择：
- 交易量>$10M的市场
- 买卖价差<0.2%
- 热门市场
```

### 技巧2：使用限价单
```
不要用市价单：
- 市价单滑点大
- 限价单控制价格
- 等待成交
```

### 技巧3：分批刷量
```
不要一次刷完：
- 分多次小额刷
- 降低市场影响
- 减少滑点
```

### 技巧4：选择合适时间
```
在交易活跃时段：
- 流动性更好
- 价差更小
- 成交更快
```
