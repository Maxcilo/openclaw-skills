### 统一工具：transfer.js ✅
创建了统一的批量转账工具，基于地址清单-0-20账号系统：
```bash
node transfer.js --balance              # 查所有余额
node transfer.js --amount 0.04 --to 1-20  # 分发到子地址
node transfer.js --collect              # 从1-20归集到0号
node transfer.js --collect --from 1-5   # 从指定编号归集
node transfer.js --dry-run              # 模拟模式
```

### 技术要点
- ethers v5（不是v6），API用 `ethers.providers.JsonRpcProvider` 和 `ethers.utils.parseEther`
- 可用RPC：`https://ethereum-sepolia-rpc.publicnode.com`（PublicNode最稳定）
- Ankr和rpc2.sepolia.org不稳定，Infura需要付费key
- 旧的 evm-batch-transfer.js 用了ethers v6 API，不兼容当前环境
