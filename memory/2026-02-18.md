## 技术细节

### prediction-log.js 用法
```bash
# 添加交易
./prediction-log.js add Simmer "市场名" Yes 0.65 100 Weather "依据"

# 查看持仓
./prediction-log.js open

# 结算交易
./prediction-log.js resolve <ID> win/loss/refund

# 查看统计
./prediction-log.js stats
```
