# 合约交易原则

## 🚨 核心教训

**血的教训：长期持有合约亏损80000u**

这是最惨痛的教训，必须永远记住！

## 📋 合约的正确定位

### 合约 = 短期工具

**用途：**
- 获得短期额外收益的机会
- 利用短期波动赚取差价
- 快进快出

**持仓时间：**
- 几小时到几天
- 最长不超过1周
- 超过1周就是错误

**特点：**
- 有资金费率（长期持有成本高）
- 有爆仓风险
- 需要时刻盯盘
- 不适合长期持有

### 现货 = 长期投资

**用途：**
- 长期投资
- 价值投资
- 不需要盯盘

**持仓时间：**
- 几周到几年
- 可以长期持有
- 不用担心爆仓

**特点：**
- 无资金费率
- 无爆仓风险
- 可以安心持有
- 适合长期投资

## ⚠️ 严格规则

### 禁止行为

❌ **绝对禁止：用合约做长期持仓**
- 不管多看好，都不要长期持有合约
- 长期看好 → 买现货
- 合约只用于短期交易

❌ **禁止：持仓超过1周**
- 超过1周就是战略错误
- 立即平仓，改用现货

❌ **禁止：不设止损的合约**
- 合约必须设止损
- 没有止损 = 等待爆仓

### 正确做法

✅ **短期交易（几小时到几天）**
- 看到机会 → 开仓
- 获利或止损 → 平仓
- 不要恋战

✅ **快进快出**
- 赚到就走
- 不要贪婪
- 保护利润

✅ **长期看好 → 买现货**
- 看好某个币种
- 用现货长期持有
- 不要用合约

## 💡 决策流程

### 开仓前问自己

**Q1: 这是短期机会还是长期看好？**
- 短期机会 → 可以用合约
- 长期看好 → 用现货

**Q2: 预计持仓多久？**
- 几小时到几天 → 可以用合约
- 超过1周 → 用现货

**Q3: 能不能时刻盯盘？**
- 能 → 可以用合约
- 不能 → 用现货

**Q4: 设好止损了吗？**
- 设好了 → 可以开仓
- 没设 → 不要开仓

### 持仓中问自己

**Q1: 持仓多久了？**
- 超过3天 → 考虑平仓
- 超过1周 → 立即平仓

**Q2: 还能盯盘吗？**
- 不能 → 立即平仓
- 能 → 继续持有

**Q3: 盈利了吗？**
- 盈利 → 考虑止盈
- 亏损 → 检查止损

## 🎯 记住这些

1. **合约 ≠ 投资工具**
   - 合约是交易工具
   - 不是投资工具

2. **时间是合约的敌人**
   - 持仓时间越长，风险越大
   - 资金费率会累积
   - 心理压力会增加

3. **80000u的教训**
   - 永远不要忘记
   - 这是用真金白银换来的
   - 不要重蹈覆辙

4. **快进快出才是王道**
   - 看到机会 → 进
   - 赚到钱 → 出
   - 不要恋战
